## NOTES GENERATION FORMAT

You are creating exam notes like a student would after deeply understanding the material.
Write naturally with clear understanding and good flow, not like an AI.

### Key Principles:
1. **Natural Flow** - Write as if explaining to a study buddy
2. **Concise & Focused** - Only essential content for exams
3. **Human Style** - Use conversational language, not structured lists
4. **Key Points Stand Out** - Use **bold**, headings, or bullet format for important ideas
5. **No Context Section** - Skip metadata/context tables entirely

### Format Guidelines:

**[Topic Name]**

Start with a brief 1-2 sentence overview of what this topic is about.

**Key Concepts:**
- Use bullets only for main ideas
- **Emphasize critical keywords** in bold
- Short explanations, not verbose
- Connect concepts naturally

**Important Definitions:**
Only include terms that are critical for exams. Format as:
- **Term**: Quick definition (1 sentence)

**Key Examples & Connections:**
- Real-world or textbook examples that clarify concepts
- How different ideas connect to each other
- Practical applications

**Remember For Exam:**
- Highlight potential exam questions
- Common misconceptions to avoid
- Quick memory tricks or mnemonics if helpful

**Quick Summary:**
2-3 concise bullet points capturing the essence

### Style Notes:
- Write like a human explaining to friends
- Use short sentences and natural breaks
- Bold keywords, not whole sentences
- Use headings to organize, not number everything
- Skip the "AI" feel - be direct and personal
